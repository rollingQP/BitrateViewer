# BitrateViewer
A lightweight Python application for visualizing video bitrate.

## Overview
BitrateViewer analyzes video files and displays their bitrate data in an intuitive graphical interface.

## Features
Multi-language Support — Available in English and Simplified Chinese.
Hover Preview — Preview video frames by hovering over the bitrate graph.
Smart CPU Scheduling — Automatically switches to efficiency cores (E-Cores) when running in the background to reduce power consumption.

## Usage
1. Download and add ```ffprobe``` in your system PATH. Or place the executable ```ffprobe``` in the ```./lib``` folder alongside video_bitrate_viewer.py
2. Run ```python3 video_bitrate_viewer.py```
